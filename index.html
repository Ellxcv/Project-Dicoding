Cara Membuat Halaman Login dengan HTML, CSS, dan JavaScript

Pada tutorial ini, kita akan membahas langkah-langkah untuk membuat halaman login yang responsif menggunakan HTML, CSS, dan JavaScript. Halaman login ini akan mencakup validasi input, styling modern, dan responsif untuk berbagai perangkat.

Langkah 1: Struktur HTML

Pertama, kita mulai dengan membuat struktur dasar HTML. Kita akan membutuhkan form dengan input untuk username dan password, serta tombol submit. 

<form>
    <div class="form-group">
        <label for="username">Username:</label>
        <input type="text" id="username" name="username" required>
    </div>
    <div class="form-group">
        <label for="password">Password:</label>
        <input type="password" id="password" name="password" required>
    </div>
    <button type="submit">Login</button>
</form>

Langkah 2: Styling dengan CSS

Selanjutnya, kita tambahkan CSS untuk memberikan gaya pada form login kita. Pastikan untuk membuat form yang responsif agar dapat terlihat baik di berbagai perangkat. 

form {
    max-width: 400px;
    margin: 0 auto;
    padding: 20px;
    background-color: #f4f4f4;
    border-radius: 8px;
    box-shadow: 0 0 10px rgba(0, 0, 0, 0.1);
}

.form-group {
    margin-bottom: 15px;
}

.form-group label {
    display: block;
    margin-bottom: 5px;
}

.form-group input {
    width: 100%;
    padding: 10px;
    border: 1px solid #ccc;
    border-radius: 4px;
}

button {
    width: 100%;
    padding: 10px;
    background-color: #4CAF50;
    color: white;
    border: none;
    border-radius: 4px;
    cursor: pointer;
}

button:hover {
    background-color: #45a049;
}

Langkah 3: Validasi dengan JavaScript

Terakhir, kita tambahkan validasi sederhana dengan JavaScript untuk memastikan bahwa pengguna memasukkan username dan password yang valid. 

document.querySelector('form').addEventListener('submit', function(e) {
    e.preventDefault();
    var username = document.getElementById('username').value;
    var password = document.getElementById('password').value;

    if (username === '' || password === '') {
        alert('Username dan password harus diisi');
    } else {
        alert('Login berhasil');
    }
});
